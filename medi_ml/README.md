# 🏥 Medi Project - Disease Prediction Module

## 📌 Overview
This module adds Machine Learning-based disease prediction to the Medi web project.

Users select symptoms → system predicts disease → suggests medicines.

---

## ⚙️ Tech Stack
- Python
- Flask (API)
- Scikit-learn (ML)
- PHP (Frontend integration)
